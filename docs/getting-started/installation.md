# Installation

Install `isometric-react` using your preferred package manager:

```bash
npm install isometric-react
```

```bash
yarn add isometric-react
```

```bash
pnpm add isometric-react
```

## Peer dependencies

`isometric-react` is intended for React 19 projects. Make sure your app uses compatible versions of:

- `react`
- `react-dom`

## Import styles

Import the bundled stylesheet once in your app entry point (for example `src/main.tsx`, `src/index.tsx`, or `src/App.tsx`):

```ts
import "isometric-react/isometric.css";
```

After the CSS is loaded, you can start rendering components immediately.
